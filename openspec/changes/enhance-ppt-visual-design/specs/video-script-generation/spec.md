# Capability: Video Script Generation

## MODIFIED Requirements

### Requirement: 视觉建议生成

系统 SHALL 在生成脚本时为每页提供视觉设计建议，指导页面生成阶段的视觉表现。

#### Scenario: 数据可视化建议
- **WHEN** 页面内容包含数字、统计数据
- **THEN** 脚本应包含数据可视化建议（如"使用柱状图展示"、"使用数字卡片突出关键指标"）
- **AND** 建议应具体、可执行

#### Scenario: 布局建议
- **WHEN** 生成页面脚本
- **THEN** 脚本应包含布局建议（如"使用左右分栏对比"、"使用卡片式网格布局"）
- **AND** 布局建议应与内容特点匹配

#### Scenario: 视觉元素建议
- **WHEN** 页面内容适合使用图标或装饰元素
- **THEN** 脚本应建议使用的视觉元素类型
- **AND** 建议应包含元素的用途和位置

### Requirement: 内容结构化

系统 SHALL 将页面内容结构化，便于页面生成阶段进行视觉设计。

#### Scenario: 数据提取
- **WHEN** 内容包含数字、百分比、统计信息
- **THEN** 脚本应将数据提取为结构化格式
- **AND** 包含数值、单位、标签等完整信息

#### Scenario: 关键点提取
- **WHEN** 生成页面脚本
- **THEN** 脚本应提取 3-5 个关键要点
- **AND** 关键点应简洁、突出重点

#### Scenario: 层级划分
- **WHEN** 内容包含多层级信息
- **THEN** 脚本应明确标注信息层级（主标题、副标题、要点、细节）
- **AND** 层级划分应清晰、合理

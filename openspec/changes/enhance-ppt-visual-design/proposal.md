# Change: 增强 PPT 视觉设计丰富度和创意表现力

## Why

当前生成的 PPT 页面存在严重的视觉单调问题：

1. **视觉元素匮乏**：页面只有纯文字和简单渐变背景，缺乏图表、图标、装饰性图形等视觉元素
2. **布局单一**：所有页面都是居中文字排列，缺乏多样化的布局方式（卡片式、分栏、网格等）
3. **数据展示能力弱**：无法展示数据可视化、统计图表、排行榜等信息
4. **设计缺乏层次感**：没有视觉层次、深度和空间感，页面扁平单调
5. **色彩运用保守**：只有简单的渐变色，缺乏丰富的配色方案和视觉冲击力

参考现代优秀 PPT 设计（如用户提供的示例），理想的页面应该包含：
- **丰富的数据可视化**：柱状图、折线图、饼图、雷达图、排行榜等
- **多样化的布局**：卡片式布局、分栏布局、网格布局、大标题+内容区等
- **视觉元素丰富**：图标、装饰性图形、色块、分割线、背景图案等
- **现代设计风格**：玻璃拟态、渐变、阴影、圆角、动态效果等
- **清晰的视觉层次**：通过大小、颜色、位置建立信息层级

## What Changes

### 核心变更

1. **增强 AI 页面设计能力**
   - 在 PAGES 步骤的 AI Prompt 中明确要求生成丰富的视觉元素
   - 提供数据可视化、图表、图标等设计指导
   - 强调多样化布局和现代设计风格

2. **扩展页面元素类型支持**
   - 支持数据图表元素（柱状图、折线图、饼图、雷达图等）
   - 支持图标和装饰性图形
   - 支持卡片、色块、分割线等布局元素
   - 支持更丰富的背景效果（渐变、图案、玻璃拟态等）

3. **优化 PPT 生成服务**
   - 移除固定的设计模板限制
   - 支持更灵活的元素组合和布局
   - 增强 CSS 样式生成能力，支持现代设计效果

4. **改进脚本生成质量**
   - SCRIPT 步骤生成时识别适合可视化的数据和信息
   - 为每页提供视觉建议（建议使用的图表类型、布局方式等）
   - 提供更详细的内容结构化信息

### 设计原则

- **内容驱动**：根据内容特点选择最合适的视觉表现形式
- **数据优先**：优先将数据和统计信息可视化
- **层次清晰**：通过视觉元素建立清晰的信息层级
- **风格现代**：采用现代设计趋势（玻璃拟态、渐变、阴影等）
- **创意自由**：每页都应该有独特的设计，避免千篇一律

## Impact

### 受影响的能力规范
- **video-page-generation**（修改）：增强页面生成的视觉设计能力
- **video-script-generation**（修改）：增加视觉建议和数据识别

### 受影响的代码
- `src/modules/workflow-steps/steps/pages.step.ts`：增强 AI Prompt，添加视觉元素指导
- `src/modules/workflow-steps/steps/script.step.ts`：增加视觉建议生成
- `src/modules/ppt/ppt.service.ts`：移除模板限制，增强元素支持
- `src/modules/ppt/ppt.types.ts`：扩展元素类型定义
- `src/modules/page-generation/page-generation.service.ts`：优化页面生成逻辑

### 非破坏性变更
- 不改变现有的数据结构和 API
- 向后兼容现有的页面生成流程
- 仅增强 AI 生成的视觉效果和元素丰富度

### 预期效果
- PPT 页面视觉丰富度显著提升
- 数据和信息展示更加直观和专业
- 页面设计更具现代感和创意性
- 用户满意度提升
